# Shop2GY Version 1.2 — Customer & Warehouse Operations MVP

Static deployment with no Node.js or build command required.

## Included
- Approved Shop2GY public website design
- Quote calculator, tracking and Shop For Me
- Customer registration, virtual locker and package center
- Delivery requests, invoice center and notifications
- Warehouse receiving workflow
- Simulated barcode scan
- Package queue and status advancement
- Shipment manifest builder
- Admin pricing controls

## Deploy on Vercel
- Framework Preset: Other
- Build Command: blank
- Install Command: blank
- Output Directory: blank

Upload these files to the root of the same GitHub repository and replace the prior versions.
