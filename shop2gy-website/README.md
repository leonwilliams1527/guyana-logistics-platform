# Shop2GY Website MVP

Shop2GY is a package forwarding, e-commerce, and last-mile delivery platform for Guyana.

## Included in this build
- Professional homepage.
- Services section.
- Smart quote calculator with product types and hazardous item warnings.
- Package tracking UI.
- Customer ID signup preview.
- Customer, admin, warehouse, and driver portal previews.
- Marketing automation center.
- Supabase database schema.
- Deployment documentation.

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Supabase-ready database
- Vercel-ready deployment

## Run Locally
```bash
npm install
npm run dev
```
