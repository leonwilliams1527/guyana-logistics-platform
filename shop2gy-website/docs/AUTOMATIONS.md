# Shop2GY Automation Specification

## Onboarding Journey
Trigger: customer creates account.
1. Welcome email.
2. Customer ID confirmation.
3. U.S. shipping address instructions.
4. How to ship your first package.
5. First shipment reminder after 7 days if no package received.
6. First shipment discount after 14 days if no package received.

## Package Notification Journey
Trigger: warehouse package status changes.
- Received in Florida.
- Invoice required.
- Quote ready.
- Payment needed.
- Paid.
- Shipped from U.S.
- Arrived in Guyana.
- Customs clearance.
- Ready for pickup.
- Out for delivery.
- Delivered.
- Review request.

## Abandonment Journeys
- Signup started but not completed.
- Account created but no first shipment.
- Quote generated but not paid.
- Cart started but purchase not completed.
- Delivery request started but not submitted.

## Internal Alerts
- New customer registered.
- High-value package received.
- Hazardous item flagged.
- Missing invoice.
- Package delayed.
- Delivery failed.
